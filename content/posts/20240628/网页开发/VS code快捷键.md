# **开发工具-VScode 快捷键**

1.Ctrl+N 新建文件

2.Ctrl+S 保存文件

3.Ctrl + 加号键，Ctrl + 减号键 可以放大缩小视图

4.快捷生成页面骨架结构

​ 输入 ! 按下 Tab 键

5.Alt + B 浏览器快捷打开

6.shift+alt+下（上）快速复制黏贴

7.选定多个相同的单词，先双击，后 ctrl+d

8.添加多个光标，ctrl+alt+上（下）

9.全局替换，ctrl+h

10.快速定位，ctrl+g
